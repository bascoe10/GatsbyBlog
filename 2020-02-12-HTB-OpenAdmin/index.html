Findings

There is an opennet admin application that is running at http://10.10.10.171/ona/. The version is 18.1.1 This version is vulnerable to RCE. This will be used for the initial foothold. With this foothold the next step is to enumerate to elevate privileges from www-data. There are 2 users accounts avialable jimmy joana

Further enumeration will lead to database_settings.inc.php that contains a password. This password can be used to log in as the user jimmy via ssh.

$ cat /opt/ona/www/local/config/database_settings.inc.php
<?php

$ona_contexts=array (
  'DEFAULT' => 
  array (
    'databases' => 
    array (
      0 =>
